# AirMIDIController
Send MIDI Signals to DAW using hand motions.

 *  COMPONENT LIST
 *  1x   Arduino Uno
 *  2x   HC-SR04 Ultrasonic sensor
 *  8x   Jumper Wires (Male to Male)
 *  1x   Breadboard

Step by Step Instructions for the project at this link:
